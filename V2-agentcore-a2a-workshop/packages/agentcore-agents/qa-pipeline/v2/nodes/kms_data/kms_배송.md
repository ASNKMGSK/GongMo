# KMS — 배송

_탭 출처: 코오롱 업무 정확도 auto_qa_criteria.xlsx / 시트 `배송` / 추출 행 3_

## [1] 기본 배송일
- **조건**: 배송 관련 문의 중, 주문/결제 건에 대한 단순한 배송 일정을 문의하였을 경우
- **필수 키워드**: 영업일 기준, 3에서 5일
- **필수 안내 사항**:
  - 결제완료 후 영업일 기준 평균 3~5일 이내에 배송되나, 재고 부족 시 매장재고 수급 후 배송하므로 추가시일이 소요될 수 있어 시간양해 부탁드립니다.

## [2] 출고 지연 문의
- **조건**: 배송 관련 문의 중, 주문/결제 건에 대한 재고 수급 중 또는 출고 보류 상태를 문의하였을 경우
- **필수 키워드**: 재고, 품절
- **필수 안내 사항**:
  - 물류센터 재고 부족으로 매장출고 가능재고 확인 후 배송될 예정으로 추가시일이 소요될 수 있으며, 품절 시 알림톡 발송되는 점 양해 부탁드립니다.

## [3] 출고 완료 배송일
- **조건**: 배송 관련 문의 중, 주문/결제 건에 대한 출고 완료가 확인된 내용의 상담이 진행될 경우
- **필수 키워드**: 송장, 문자
- **필수 안내 사항**:
  - 출고 후 영업일 기준 3-5일 이내 배송되나 택배사 사정에 따라 변동될 수 있습니다. 출고 후 상품의 위치는 운송장 조회를 통해 확인 가능하며, 고객님께 문자 발송 도와드리겠습니다.
