## 2. 주요 판정 원칙 (kolon_v1)

- **#1 첫인사**: "반갑습니다 / 안녕하세요" + "코오롱 고객센터" + 상담사명 3요소.
- **#2 끝인사**: "다른 문의사항 있으실까요" 추가 문의 확인 + 인사말/상담사명 중 1개만 있어도 3점 (2-요소 완화).
- **#3 경청**: 2026-04-21 평가표에서 제거 (STT 한계). 5점은 #15 로 이관.
- **#6 정중한 표현**: 반말/사물존칭 ("되시겠습니다", "나오시겠습니다") 1회 이상 시 감점.
- **#7 쿠션어**: refusal-gated — 재고 부족/교환비 발생/반송장 누락 경고 등 거절/양해 상황에서만 판정.
- **#10 설명 명확성**: 교환비 (자사 무상 / 타사 5천원), 영업일 (2~3일), 합포장 조건 등 숫자/조건 명시 여부.
- **#13 부연 설명**: 자사 상품 1회 무상 교환 정책, 반송장 보관 의무, 합포장 반송 가능 여부 선제 안내.
- **#15 정확한 안내**: `business_knowledge/` 매뉴얼 RAG 필수. 매뉴얼 부재 시 `unevaluable` 반환. 스텝 `[15, 10, 5, 0]` (2026-04-21 확장).
- **#16 필수 안내**: intent=환불취소 시 반송장 보관, 회수 기사 방문 일정, 외부 착용/택 제거 여부 확인 3종.
- **#17/#18 개인정보**: 고객 주소/주문번호를 상담사가 먼저 발화하면 감점. force_t3.

## 3. 카테고리 전체 영향 (Critical)

- 고객 주소/연락처/주문번호 상담사 선발화 → #17/#18 = 0 + 컴플라이언스 보고
- 반말/비하/욕설/업무회피 → 총점 0점
- 매뉴얼 대비 허위 안내 미정정 (예: 교환비 무료라고 잘못 안내 후 정정 없음) → #15 = 0
