## 교환 기본 정책

- **자사 상품 1회 무상 교환**: 코오롱몰 자체 상품 (코오롱스포츠, 헨리코튼, 넥스, 왁, 골든베어 등) 은 1회에 한해 왕복 택배비 무상 교환 가능.
- **교환 가능 조건**:
  - 외부 착용 흔적 없음
  - 텍(tag) 제거되지 않음
  - 세탁 이력 없음
  - 박스/부속품 포함 (비닐 포장은 훼손되어도 무방)
- **교환 가능 기간**: 상품 수령일 기준 7일 이내 접수 (수령일 + 7 영업일).
- **2회차 이상 교환**: 왕복 택배비 5,000원 고객 부담 (환불 예정 금액에서 차감 가능).
- **접수 → 회수 → 재배송**:
  - 접수일로부터 영업일 기준 2~3일 내 회수 기사 방문
  - 회수 상품 입고 후 검수 영업일 2~3일
  - 검수 통과 시 재배송 출고, 재고 여부에 따라 추가 영업일 소요

**source_ref**: kolon-exchange-policy v1.0

## 반품 기본 정책

- **반품 사유 구분**:
  - **단순 변심 / 사이즈·색상 불만**: 반품 택배비 2,500원 고객 부담 (환불 예정 금액에서 차감).
  - **상품 불량 / 오배송 / 포장 불량**: 반품 택배비 전액 코오롱몰 부담. 검수 결과 실제 불량 판정 시 적용.
- **반품 가능 기간**: 상품 수령일 기준 7일 이내 접수.
- **반품 불가 조건**:
  - 외부 착용, 텍 제거, 세탁 이력 있는 경우
  - 향수 / 화장품 / 속옷 등 위생 상품 개봉 후
  - 고객 과실에 의한 훼손
- **환불 금액 산정**:
  - 결제 금액 전액 환불 (단순 변심인 경우 택배비 2,500원 차감)
  - 사용 포인트 / 마일리지는 반품 처리 완료 후 복구 (실제 복구 소요: 영업일 1~2일)
  - 카드 결제는 카드사 취소 처리 (영업일 3~5일 소요)
- **입점사 상품**: 입점사(브랜드사) 판매 상품은 입점사 정책에 따름 — 반드시 입점사 확인 후 재연락 안내.

**source_ref**: kolon-return-policy v1.0

## 상품 불량 판정 기준 및 검수 절차

- **불량 판정 대상**:
  - 제조 과정상 결함 (박음질 풀림, 원단 구멍, 프린팅 불균일)
  - 배송 중 훼손 (찢김, 오염)
  - 상세페이지 표기와 상이한 원단 / 색상 / 사이즈 (실측 오차 ±2cm 초과)
  - 포장 불량으로 상품 가치 훼손 (선물 용도 부적합 포함)
- **고객 주관 불만 (단순 변심에 해당)**:
  - "느낌이 다르다", "사진과 색감이 약간 차이", "원단 질감이 예상과 다름" 등 주관적 사유는 원칙적으로 **단순 변심**으로 분류.
  - 단, 고객이 강하게 불량을 주장하는 경우 **교환 우선 진행** 후 재수령 시 동일 문제면 반품 전환 안내.
- **검수 절차**:
  1. 상품 입고 후 물류센터 검수 (영업일 2~3일)
  2. 불량 확정 시: 고객 부담 배송비 환원, 환불 금액에 전액 반영
  3. 불량 아님 확정 시: 폐기 처분 또는 고객 재수령 중 선택 안내 (추가 배송비 발생)
- **검수 결과 통지**: SMS 또는 유선 연락. 분쟁 시 물류센터 검수 사진 제공 가능.

**source_ref**: kolon-quality-inspection v1.0

## 사이즈 교환 절차 및 핏 안내

- **사이즈 교환 접수 시 필수 확인**:
  1. 주문번호 / 상품명 / 현재 사이즈
  2. 교환 희망 사이즈
  3. 텍 / 외부 착용 / 세탁 여부
  4. 회수 주소 (기존 배송지와 동일 여부)
- **같은 상품 같은 사이즈 표기라도 핏에 따라 실측 상이**:
  - **레귤러핏**: 기본 사이즈 표기대로
  - **슬림핏**: 한 치수 크게 주문 권장
  - **오버핏**: 한 치수 작게 주문 권장 또는 평소 사이즈
- **사이즈 추천 불가**: 상담사는 상품 실물을 보지 않고 상세페이지만 확인 가능하므로 **구체적 사이즈 추천은 진행하지 않음**. 상세페이지의 실측 사이즈 표 안내만 가능.
- **재고 소진 가능성**: 교환 희망 사이즈가 소량 재고인 경우 교환 작업 도중 품절 가능성 안내 필수. 품절 시 자동 환불 처리 동의 확인.
- **회수 및 재배송**: 회수 후 재배송까지 영업일 기준 4~6일 소요.

**source_ref**: kolon-size-exchange-guide v1.0

## 색상 교환 절차 및 재고 제한

- **색상 교환 접수 시**: 사이즈 교환과 동일 절차. 교환 희망 색상 재고 확인 후 접수.
- **극소량 재고 색상**: 접수 시점 재고가 3개 이하인 경우 **품절 가능성 사전 고지 필수**.
- **품절 자동 환불**: 교환 작업 중 품절되면 자동으로 반품(환불) 처리로 전환되며 고객에게 SMS 통지.
- **색상 교환 가능 범위**: 동일 상품 / 동일 사이즈 내 다른 색상만 허용. 다른 상품으로의 교환은 반품 후 재주문.
- **맞교환(동시 배송/회수)**: 원칙적으로 어려우나, 재고 부족 + 포인트/마일리지 복구 곤란 등 예외 상황에서 예외적 접수 가능. 단, 회수와 재배송이 별도 진행되므로 포장 이원화 필요.

**source_ref**: kolon-color-exchange-guide v1.0

## 주문 취소 절차

- **배송 전 취소** (출고 전):
  - 홈페이지 마이페이지 → 주문 상세 → 취소 버튼으로 즉시 취소 가능
  - 취소 완료 시 결제 수단에 따라 즉시 취소 또는 영업일 3~5일 내 환급
- **배송 중 취소** (출고 후 ~ 수령 전):
  - 이미 출고된 경우 고객센터 상담으로만 취소 접수 가능
  - 배송 완료 후 반품 절차로 전환 안내
- **부분 취소**:
  - 여러 상품 중 일부 취소 시 해당 상품 별도 접수
  - **입점사 혼합 주문의 경우 부분 취소 제한** — 홈페이지에서 하나만 반품/취소가 불가한 경우 고객센터 유선 접수 필요
- **재고 확인 지연에 따른 취소**: 주문 후 영업일 2일 이상 출고 지연 시 고객 요청에 따라 주문 취소 및 전액 환불.
- **포인트 / 마일리지 복구**: 주문 취소 완료 후 복구 (영업일 1~2일 이내).

**source_ref**: kolon-order-cancel-policy v1.0

## 부분 반품 / 부분 교환 처리

- **여러 상품 중 일부만 반품 / 교환** 요청 시:
  - 상담사 유선 접수 필수 (홈페이지 마이페이지에서 부분 반품 버튼 미지원 건 존재)
  - 반품 대상 상품의 주문번호 / 상품명 / 수량 확인
  - 반품 사유 각각 기재
- **택배비 산정**:
  - 부분 반품이라도 왕복 택배비 2,500원 (단순 변심) 또는 무료 (불량) 적용 기준 동일
  - 여러 상품을 하나의 박스로 반송해도 택배비는 1회 기준
- **혼합 주문 (자사 + 입점사)**:
  - 자사 상품과 입점사 상품이 섞인 주문은 부분 반품 시 각각 별도 회수 진행
  - 입점사 상품은 입점사 회수 기사가 별도 방문
- **부분 교환 후 잔여 상품 처리**: 교환 대상 외 상품은 그대로 사용되는 것으로 기록.

## 반송장 발급 및 보관 안내

- **반송장 발급 방식**: 회수 기사가 방문 시 반송장을 **직접 지참하여 전달**. 고객이 별도 출력 불필요.
- **반송장 보관 의무**:
  - 반송장은 **환불 완료 시까지 반드시 보관** — 분실 시 상품 추적 불가 → 환불 지연 또는 불가
  - 회수 기사가 두고 간 반송장 사본은 버리지 말 것
- **반송장 번호 확인**: 회수 완료 후 문자 안내. 고객이 직접 택배사 사이트에서 배송 조회 가능.
- **상품 포장**:
  - 원래 박스가 없는 경우 **일반 택배 박스 / 아무 박스 사용 가능**
  - 비닐만 있는 경우라도 회수 기사에게 전달 가능
  - 다만 상품 훼손 방지를 위해 가능한 한 견고한 포장 권장

**source_ref**: kolon-return-label-guide v1.0

## 회수 기사 방문 일정 및 미방문 대응

- **방문 일정**:
  - 접수일 기준 **영업일 2~3일 이내** 방문 (토/일/공휴일 제외)
  - 당일 접수 건은 당일 방문 불가 — 다음 영업일 이후
  - 토요일 방문: 지역별 토요 운영 기사에 한해 가능. 미운영 지역은 월요일 방문
- **방문 전 연락**:
  - 회수 기사가 방문 당일 오전 중 SMS 또는 유선 연락
  - 부재 시 재방문 일정 별도 조율
- **미방문 발생 시**:
  - 접수일 + 영업일 3일 경과 후에도 미방문 시 **고객센터로 재연락 요청**
  - 고객센터에서 회수 기사 재배정 접수 진행 (재접수 시 재회수일 영업일 2~3일 추가)
- **회수 장소 지정**:
  - 현관 앞 / 경비실 / 택배함 등 수령 장소 지정 가능
  - 상품 분실 방지를 위해 가능하면 대면 전달 권장

## 포장 불량 / 선물 용도 처리

- **포장 불량 판정 기준**:
  - 비닐 포장의 스티커 중복 부착 / 끈적임 흔적
  - 박스 케이스 누락 또는 파손
  - 선물용 케이스(종이 박스) 미포함
  - 제품 자체에는 이상 없으나 선물로 사용 불가한 수준
- **처리 방법**:
  1. 단순 변심이 아닌 **포장 불량 사유**로 접수 (택배비 무료)
  2. 입점사 판매 상품인 경우 케이스 재발송 가능 여부 **입점사 확인 후 재연락**
  3. 입점사 확인이 필요한 경우 고객 통화 가능 시간대 확인 후 재연락 약속
- **재포장 발송** (가능한 경우):
  - 케이스만 별도 발송 가능 시 영업일 2~3일 내 배송
  - 케이스 미제공 시 반품 처리로 전환 안내
- **입점사 연락 시 기록**: 고객의 포장 불량 내용은 상담 기록에 상세히 남겨 입점사 문의 시 근거 자료로 사용.

**source_ref**: kolon-packaging-policy v1.0

## 재고 품절 / 대체 상품 안내

- **품절 시점 구분**:
  - **접수 시점 품절**: 교환 접수 자체 불가. 반품(환불) 또는 다른 색상/사이즈 제안.
  - **교환 작업 중 품절**: 회수 완료 후 재배송 직전 품절된 경우 → **자동 환불 처리**.
  - **주문 후 재고 없음 확인**: 주문 → 출고 준비 중 재고 불일치 확인 시 전액 환불 + SMS 통지.
- **자동 환불 처리**:
  - 결제 수단으로 자동 환급
  - 사용 포인트 / 마일리지 자동 복구
  - SMS 통지 발송
- **대체 상품 안내**:
  - 유사 상품이 있는 경우 상세페이지 링크 안내 (단, 강권하지 않음)
  - 재입고 예정일 문의 시 해당 브랜드 담당자 확인 후 재연락 (대개 확정 불가)
- **재입고 알림 서비스**: 상품 상세페이지에서 재입고 알림 신청 가능 안내.

## 본인 확인 절차

- **필수 확인 요소 (2-요소)**:
  1. 고객 성함 (풀네임)
  2. 주문 시 등록된 휴대폰 번호
- **확인 절차**:
  - "성함과 연락처 말씀 부탁드립니다"
  - 고객 답변 후 "***고객님 본인 맞으십니까?" 재확인
  - "소중한 정보 확인 감사드립니다" 완료 멘트
- **본인 확인 불일치 시**:
  - 주문자와 통화자가 다른 경우 (가족 대리 문의 등) 주문자 본인의 구두 위임 또는 재통화 요청
  - 개인정보 관련 문의 (주소 변경, 포인트 조회 등) 는 반드시 주문자 본인만 가능
- **보안 주의**:
  - 주민등록번호 / 신용카드 번호 / 비밀번호는 요구하지 않음
  - 전화번호 / 이름 외 추가 개인정보 요청 금지

## 상담 녹취 및 시작 안내

- **의무 녹취 안내**:
  - 상담 품질 향상 및 분쟁 예방을 위해 모든 상담은 녹취됨
  - 녹취 파일은 관련 법령에 따라 보관
- **상담 시작 표준 멘트**:
  - "반갑습니다, 코오롱 고객센터 [상담사명]입니다. 무엇을 도와드릴까요?"
- **상담 종료 표준 멘트**:
  - "다른 문의사항 있으실까요?" (추가 문의 확인)
  - "감사합니다. 코오롱 상담사 [상담사명]이었습니다." (상담사 재고지)
- **상담 운영 시간**:
  - 평일 09:00 ~ 18:00 (KST)
  - 점심시간 12:00 ~ 13:00 (인입 가능하나 대기 증가)
  - 토요일 / 일요일 / 공휴일 휴무
  - 야간 / 휴무일: 홈페이지 1:1 문의 또는 챗봇 이용 안내

## 브랜드별 정책 구분 (자사 / 입점사)

- **자사 브랜드** (코오롱몰 직접 운영):
  - 코오롱스포츠 (KOLON SPORT)
  - 헨리코튼 (HENRY COTTON'S)
  - 넥스 (NEX)
  - 왁 (WAAC)
  - 골든베어 (GOLDEN BEAR)
  - 시리즈 (SERIES)
  - 럭키슈에뜨 (LUCKY CHOUETTE)
  - 공통 규정: 1회 무상 교환, 반품 택배비 2,500원 (단순 변심), 회수 영업일 2~3일
- **입점사 브랜드** (제휴 판매):
  - 입점사 자체 정책에 따라 교환 / 반품 조건 상이
  - 택배비 / 회수 일정 / 검수 기준 모두 입점사 정책 적용
  - 고객센터에서 1차 접수 후 **입점사 확인 → 재연락** 프로세스
- **브랜드 확인 방법**:
  - 상품 상세페이지 하단 "판매자" 정보로 구분
  - "코오롱인더스트리(주)" 표기 시 자사, 그 외 업체명은 입점사
- **복수 브랜드 주문**: 자사 + 입점사 혼합 주문은 각각 별도 회수 / 재배송 진행.

## 골프웨어 핏 분류 및 사이즈 안내

- **릴렉스핏**: 골프 스윙 시 어깨·팔 가동성 우선. 일반 의류 대비 **0.5~1 사이즈 작게** 권장.
- **슬림핏**: 라운딩 시 실루엣 강조. 평소 사이즈와 동일 또는 1 사이즈 크게.
- **오버핏**: 트렌디 캐주얼 라인. 평소 사이즈 그대로 또는 1 사이즈 작게.
- **브랜드별 차이**:
  - **말본 (Malbon)**: US 사이징. 한국 사이즈 + 1단계 (예: M → L)
  - **골든베어 골프**: 한국 표준 사이징. 평소 사이즈 동일.
  - **디아도라**: 슬림핏 위주. 1단계 크게 권장.
  - **부티커**: 여성 라인 중심. 한국 사이즈 동일.
  - **지포어 (G/Fore)**: US 사이징, 미국 핏 큼. 1~2 단계 작게.
- **사이즈 교환 우선 안내**: 골프웨어는 사이즈 교환 비중이 일반 의류보다 30%p 이상 높음 → 첫 안내 시 핏 정보 적극 제공.

## 골프 시즌 기획전 및 한정 컬러

- **시즌 운영**: 골프 성수기 (3월~10월) 중심. 비시즌 (11월~2월) 은 방한 라인업 + 실내 골프장 의류 강조.
- **한정 컬러 정책**:
  - 시즌 한정 컬러 (예: 말본 핑크 라인, 디아도라 미국 독립기념일 컬렉션) 는 **재입고 불가**.
  - 사이즈/색상 교환 요청 시 동일 컬러 재고 없으면 **반품 후 재주문** 안내.
  - 한정 컬러 환불은 일반 환불 정책 동일 (1주일 이내).
- **신상 출시 주기**: 매월 둘째/넷째 주 목요일 출시. 사전 알림 신청 가능.
