# cartgolf 인바운드 예약팀 (샘플)

라운딩 예약/변경/취소 응대 팀 샘플 폴더. 실제 운영 시 `golden_set/`, `tenant_config.yaml` 등을 이 폴더에 배치하면 예약 문의 상담에 한정해 우선 적용됩니다.

## 예약팀 특화 포인트

- **#15 정확한 안내**: 예약 가능 시간 · 요금 · 규정 정확도 중요 (Business Knowledge 비중 상향 권장)
- **#16 필수 안내**: 취소 수수료, 날씨 취소 정책 고지 의무
- **#8 니즈 파악**: 원하는 티오프 시간대 / 동반자 수 / 카트 필요 여부 등 재질문 필수
