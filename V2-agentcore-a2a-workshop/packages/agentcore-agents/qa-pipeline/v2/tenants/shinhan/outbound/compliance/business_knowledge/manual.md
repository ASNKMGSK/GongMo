## 금소법 제19조 설명의무 6대 항목

- **법적 근거**: 금융소비자보호법 제19조 (설명의무)
- **적용 대상**: 신용카드 가입을 포함한 모든 금융상품 권유·판매 행위
- **6대 핵심 설명 항목** (모두 가입 의사 결정 전 설명 의무):
  1. **상품명·종류**: 신용카드 / 체크카드 / 부가서비스 등 상품 분류
  2. **연회비**: 금액 + 면제 조건 (예: "전월 30만 원 이상 사용 시 면제, 미달 시 1만 원 청구")
  3. **이자율**: 할부 / 현금서비스 / 리볼빙 각각의 구간별 이율 (구체 % 명시)
  4. **수수료**: 현금서비스 수수료 / 해외 사용 수수료 / 분실 재발급 수수료 등
  5. **혜택**: 할인·적립률 / 한도 / 적용 조건
  6. **위험**: 연체 시 신용평점 영향 / 거래정지 / 추가 수수료 / 채무 부담 가중 가능성
- **이행 판정 기준**:
  - 6대 항목 모두 가입 의사 결정 전 발화 시점 명확 → **이행** (15점)
  - 일부 항목 모호하거나 "약관 참조" 로 회피 → **부분 이행** (10점)
  - 절반 미만 또는 위험 적극 부정 → **미이행** (0점)
- **금지 패턴**:
  - "약관에 다 적혀 있으니까 가입 후 보세요" (회피)
  - "위험 같은 거 신경 안 쓰셔도 돼요" (적극 부정)
  - "그건 가입 후 안내드릴게요" (가입 전 고지 의무 회피)

## 금소법 적합성 원칙 — 취약계층 정의 및 보호

- **법적 근거**: 금융소비자보호법 제17조 (적합성 원칙)
- **취약계층 정의** (단일 또는 복합 신호 식별 시):
  1. **고령자**: 만 65세 이상
  2. **이해 곤란자**: 동일 정보 3회 이상 반복 질문 / 상품 개념 자체 모호 ("신용카드가 뭐예요" 등)
  3. **장애 의심 발화**: "잘 안 들리는데" / "천천히 말씀해 주세요" 반복 / 청력·인지 곤란 신호
  4. **기타**: 기초생활수급자 / 장애인 등 본인 자진 신고
- **취약계층 식별 시 보호 조치 3종** (모두 이행해야 적합성 원칙 충족):
  1. **가족 동참 권유**: "가족 중에 함께 결정하실 분이 계시면 같이 들어주셔도 좋고"
  2. **검토 시간 보장**: "충분히 생각하신 후 결정해 주셔도 됩니다" — 즉시 결정 강요 금지
  3. **재설명 보장**: "이해가 안 되시면 언제든 다시 여쭤봐 주십시오" + 천천히 발화
- **표준 식별 멘트**:
  > "고객님 연세 여쭤봐도 될까요" (식별 시도) → 65세 이상 응답 시 보호 조치 3종 즉시 이행
- **금지 패턴**:
  - 취약 신호 인지 후에도 "일단 가입 진행할게요" 일방 처리
  - 반복 질문 발생 시 재설명 회피 ("그냥 가입하시면 됩니다")
  - 가족 동참 권유·검토 시간 보장 누락
- **위반 시 영향**: 즉시 0점 + #919, #921, #922 동시 위반 가능

## 금소법 제21조 부당권유 금지 패턴

- **법적 근거**: 금융소비자보호법 제21조 (부당권유 행위의 금지)
- **부당권유 4대 패턴**:
  1. **단정적 판단 제공**: "무조건 이득", "확실히 좋다", "후회 안 한다" — 미래 효용을 단정
  2. **허위·과장 표현**: "거의 모든 고객이 만족", "다른 카드는 다 안 좋고", "100 퍼센트 혜택"
  3. **불이익 사실 누락**: 위험·한도·조건·예외 사항을 의도적으로 미고지하거나 부정
  4. **강요·압박**: 시간 압박 ("지금 안 하시면 못 가입") / 거절 회피 / 반복 권유
- **이행 판정 기준**:
  - 4대 패턴 모두 부재 + 사실 기반 + 한도/위험/불이익 명시 → **이행** (5점)
  - 일부 패턴 발생 또는 한도/위험 누락 → **부분 이행** (3점)
  - 다수 패턴 동시 발생 또는 위험 적극 부정 → **미이행** (0점)
- **금지 표현 표본**:
  - "무조건 이득", "절대 후회 안 하세요"
  - "다른 카드는 다 별로고 우리만 좋아요"
  - "위험 그런 일 절대 안 일어나요"
  - "지금 결정 안 하시면 나중에 못 가입하세요"
  - "거의 모든 고객이 만족"
- **권장 대안 표현**:
  - "본인의 사용 패턴에 따라 혜택 체감이 다를 수 있습니다"
  - "결정은 충분히 검토하신 후 자율적으로 부탁드립니다"
  - "연체 시 신용평점이 하락할 수 있으며 추가 수수료가 발생할 수 있습니다"

## 금소법 제46조 청약철회권 (기간·방법·이해도 확인)

- **법적 근거**: 금융소비자보호법 제46조 (청약철회권)
- **청약철회 가능 기간**: 가입 신청일로부터 **14일 이내**
- **철회 시 효과**: 위약금 / 손해배상금 / 수수료 청구 불가
- **철회 방법** (모두 안내 의무):
  1. 1577 신한카드 고객센터 전화
  2. 신한카드 홈페이지 마이페이지 → 발급 신청 내역 → 청약철회
  3. 신한카드 앱 마이페이지 → 발급 내역 → 청약철회
- **이해도 확인 절차** (제46조 + 제19조 결합):
  1. 안내한 핵심 항목 (연회비 / 이자율 / 청약철회 14일 등) 항목별 이해 여부 재확인
  2. "안내드린 내용 이해하셨습니까" 명시적 질의
  3. "가입에 동의하시는지요" 명시적 질의 → 고객의 명시적 긍정 답변 ("네 동의합니다")
- **표준 멘트**:
  > "본 카드 가입은 금융소비자보호법 제46조에 따라 14일 이내 청약철회권 행사가 가능합니다. 별도 사유 없이도 가능하며 위약금이나 손해배상금이 청구되지 않습니다. 철회는 1577 신한카드 고객센터 또는 홈페이지 마이페이지에서 가능합니다. 안내드린 내용 모두 이해하셨고 가입에 동의하시는지요?"
- **금지 패턴**:
  - "마음에 안 드시면 안 쓰시면 돼요" (청약철회권을 단순 미사용으로 오도)
  - "안 쓰시면 알아서 해지될 거예요" (자동 해지 허위 안내)
  - 가입 의사 명시적 재확인 누락 ("일단 진행할게요" 일방 통보)
  - 항목별 이해도 검증 절차 누락

## 불완전판매 점검 워크플로 (CRM 부서 통화 사후 검수)

- **점검 범위**: 신한카드 CRM부 / 텔레마케팅부 등 통신판매 outbound 통화의 녹취 자료
- **점검 트리거**:
  - 정기 샘플링 (전체 통화의 5%)
  - 고객 민원 발생 콜 100% 점검
  - 취약계층 대상 통화 100% 점검 (시스템 식별 또는 상담사 자진 분류)
  - 가입 후 7일 이내 청약철회 발생 콜 100% 점검
- **점검 절차**:
  1. 녹취 STT 변환 → transcript 확보

  3. ★ 항목 (#919 설명의무) 부분 감점 또는 0점 시 시니어 점검자 재검토

## 설명의무 5요소 표준 스크립트 (#919 직결)

판매 통화에서 다음 5요소를 **누락 없이 + 정확한 수치/조건과 함께** 안내해야 한다. QA #919 설명의무 이행 ★ 만점 기준 (금소법 §19).

- **5요소 체크리스트**:
  1. **상품명 / 종류**: "OO카드 (신용카드 / 체크카드 / 부가서비스)" — 정확한 상품 풀네임
  2. **연회비 / 회비 구조**: "연회비 OOO원 / 1년차 면제 / 매년 청구일 D-7 사전 통지"
  3. **이자율 / 수수료**: "할부 N.N%~M.M% / 현금서비스 N.N% / 리볼빙 N.N%" 구간별 명시
  4. **혜택 / 조건**: "OO 분야 N% 적립 / 월 한도 OOO원 / 적립 조건 (전월 실적 OOO원 등)"
  5. **위험 · 제한사항**: "연회비 미납 시 부가서비스 중단 / 신용평점 영향 / 청약철회 14일"
- **표준 발화 예시**:
  - "이번에 권유드리는 OO카드는 신용카드 상품으로 연회비 OOO원, 첫해 면제 후 매년 OO월에 청구됩니다."
  - "주요 혜택은 OO 분야 N% 적립이고 전월 실적 OOO원 이상 시 적용됩니다."
  - "할부 수수료는 12개월 기준 약 N.N% 수준이며, 현금서비스 이용 시 별도 이자가 가산됩니다."
  - "위험·제한사항으로는 연회비 미납 시 부가서비스 중단, 가입 후 14일 이내 청약철회 가능하시다는 점 안내드립니다."
- **이해 확인 절차** (#922 연계):
  - "지금까지 안내드린 내용 이해되셨을까요?"
  - 고객 명시적 동의 필수 ("네, 이해했습니다" / "알겠습니다")
  - "음...", "글쎄요" 등 모호한 응답 시 재설명 의무
- **부분 점수 트리거**:
  - 5요소 중 1개 누락: 10점
  - 5요소 중 1~2개 누락 + 이해 확인 절차 미흡: 5점
  - 3개 이상 누락 또는 유리 조건만 강조 (위험·제한 누락): 0점

## 설명의무 위반 즉시 정정 표준 (#919 부분 점수 회복)

설명 도중 위험·제한사항을 누락하거나 유리 조건만 강조한 경우 통화 종료 전 정정 시 부분 점수 (5점) 인정.

- **정정 트리거**:
  - 본인 인지 (5요소 중 누락 발견)
  - 고객 질문이 누락 항목과 직결 ("연회비는요?", "수수료는요?")
- **정정 표준 발화**:
  - "추가로 안내드릴 부분이 있어 말씀드립니다."
  - "방금 혜택 부분만 안내드렸는데, 위험·제한 부분도 함께 안내드려야 정확합니다."
