## VOC 카테고리 분류 체계

신한카드 소비자보호부는 모든 민원을 다음 카테고리 체계에 따라 분류한다 (사내 표준 v3.2).

- **상품 / 서비스 분류 (Level 1)**:
  - **결제 / 청구**: 결제 오류 / 이중청구·중복승인 / 청구서 오류 / 자동이체 오류 / 환불 처리 지연
  - **카드 발급 / 한도**: 발급 거절 사유 / 한도 부족 / 한도 인하 / 발급 지연
  - **부가 서비스**: 보험 자동가입 / 멤버십 / 컨시어지 / 포인트 / 마일리지
  - **연체 / 채권**: 추심 절차 부당 / 안내 미흡 / 신용 영향 분쟁

- **운영 / 응대 분류 (Level 1)**:
  - **직원 응대**: 부당 언행 / 응대 지연 / 정보 누락
  - **시스템**: 앱 오류 / 인증 실패 / SMS 미수신
  - **개인정보**: 정보 유출 / 동의 없는 처리 / 마케팅 수신

- **분류 원칙**:
  - 복수 카테고리에 걸친 경우 모두 등록 (예: \"이중청구 + 환불 지연\" 동시 등록)
  - 하위 카테고리 (Level 2) 는 본문 키워드로 자동 추론 후 상담사 검증

## 민원 처리 SLA 및 이관 기준

- **처리 시한 (SLA)**:
  - **단순 문의**: 즉시 응답 (분류 결과 단순문의로 확정 시)
  - **일반 민원**: 영업일 기준 7일 이내 결과 통지
  - **중대 민원** (부당 응대 / 정보 유출 / 1만원 이상 손해 발생): 영업일 기준 3일 이내 + 본부 보고
  - **이의 신청**: 1차 결과에 불복 시 영업일 14일 이내 재심

- **이관 기준**:
  - **결제팀**: 결제 / 청구 / 정산 관련 (이중청구 / 매입 미취소 등)
  - **심사발급팀**: 발급 거절 사유 / 신용평가 이의
  - **컬렉션팀**: 연체 / 추심 절차 분쟁
  - **본부 소비자보호 매니저**: 중대 민원 / 본인 요청 시 즉시 연결
  - **외부 이관**:
    - 금융감독원 분쟁조정 (1332): 합의 불성립 시 안내
    - 한국소비자원 (1372): 일반 소비자 분쟁
    - 신용회복위원회 (1600-5500): 채무조정 관련

- **민원번호 발급**:
  - 모든 민원 등록 시 \"SH-YYYY-***-XXXX\" 형식으로 즉시 발급
  - 고객에게 즉시 안내 (구두 + SMS)
  - 추가 문의 시 같은 담당자 연결 가능

**source_ref**: shinhan-voc-sla v1.0

## 소비자보호기준 — 부당 응대 / 2차 가해 정의

금융감독원 \"금융소비자 보호 모범기준\" 및 신한카드 사내 소비자보호기준에 따른 정의이다. 위반 시 #916 즉시 0점 + critical_violation 트리거.

- **부당 응대 정의**:
  - **반말 / 비하 / 모욕**: \"본인이 잘못 이해\" / \"이렇게까지 화 안 내시거든요\" / 인격 폄하
  - **강요 / 압박**: \"빨리 진행\" 반복 / 다른 고객 대기 압박 도구화 / 통화 종료 강요
  - **책임 전가**: \"본인이 골라서\" / \"가맹점 잘못이라서 어쩔 수 없다\"
  - **차별 / 편견**: 연령 / 성별 / 국적 / 장애를 근거로 한 응대 차등
  - **무시**: 고객 의사 묵살 / 동일 안내 반복으로 회피

- **2차 가해 (Secondary Harm)**:
  - 민원인의 정신적 피해를 가중시키는 모든 발화 / 행위
  - 예시: 비교 압박(\"다른 분들은 안 그래요\") / 흥분 상태 라벨링(\"흥분하셔서 이상하게 들리시는 거\") / 책임 떠넘기기
  - 적발 시 본부 보고 + 상담사 재교육 + 사례에 따른 징계

- **취약계층 보호 (강화 기준)**:
  - 고령(만 65세 이상) / 장애 / 외국인 / 한국어 미숙 고객에 대한 응대 시 강화된 의무
  - \"천천히 / 명확하게 / 반복 안내\" 원칙
  - 본인 동의 하에 가족 동행 통화 가능

## 사후 연락 안내 의무 및 본인 권리

- **사후 처리 통지**:
  - 민원 처리 완료 시 SMS + 알림톡 (신한플레이) 동시 통지
  - 통지 내용: 민원번호 / 처리 결과 / 추가 문의 채널 / 만족도 조사 안내

- **만족도 조사**:
  - 일반 민원 처리 후 영업일 7일 후 SMS 발송
  - 응답률 / 만족도 결과는 분기별 본부 보고 / 상담사 KPI 반영
  - 불만족 응답 시 본부 매니저 직접 재접촉

- **녹취 사본 요청 권리**:
  - 본인 통화 녹취는 30일 보관 (분쟁 시 90일 연장)
  - 본인이 요청 시 사본 제공 (이메일 mp3 / 텍스트 전사)
  - 안내 의무: 통화 시작 시 \"녹취 됨\" + 본인 권리(사본 요청 가능) 명시

- **재접촉 채널**:
  - 동일 담당자 연결: 민원번호 알리면 자동 라우팅
  - 본부 매니저 직통: 1544-7000 → 9번 (소비자보호 매니저)
  - 외부 채널: 금감원 1332 / 소비자원 1372 (합의 불성립 시 안내 의무)

## 공감·사과 표준 응대 스크립트

상담사가 사용해야 할 표준 공감/사과 발화 패턴이다.

- **감정 라벨링 패턴**:
  - \"정말 [놀라셨겠어요 / 당황스러우셨겠어요 / 속상하셨겠어요]\"
  - \"[화가 나실 / 답답하실] 만한 상황입니다\"
  - 감정 단어 반복 사용 금지 — 고객 표현에 맞춰 변형

- **사과 패턴**:
  - 1차 사과: \"[불편 / 당황 / 걱정] 끼쳐드려서 정말 죄송합니다\"
  - 책임 인정: \"이런 일이 생긴 자체가 저희 잘못이고요\"
  - 형식적 \"네 죄송합니다\" 단독 사용 금지 (감정 라벨링 + 사과 결합 필수)

- **진정 멘트**:
  - \"천천히 말씀해 주세요 시간 충분히 드리겠습니다\"
  - \"부담스러우시면 잠시 휴식 후 재연결도 가능하시고요\"
  - 강요 금지: \"진정하세요\" / \"빨리 말씀하세요\" — 부당 응대 트리거

- **본인 권리 멘트**:
  - \"본인 의사 무시되거나 강요받는 일 절대 없으시고요\"
  - \"본부 소비자보호 매니저 직접 연결도 도와드릴 수 있습니다\"
  - \"본인 권리에 따라 녹취 사본 요청도 가능하십니다\"

## VOC 분류 정확성 5요소 (#913 직결)

상담사는 민원을 접수할 때 다음 5요소를 누락 없이 + VOC 카테고리에 정확히 매핑해야 한다. QA #913 민원 유형 분류 및 원인 파악 ★ 만점 기준.

- **5요소 체크리스트**:
  1. **VOC 카테고리**: BK-SHV-001 의 분류체계 (1차/2차/3차) 와 일치하는 라벨 명시
  2. **원인 분류**: 시스템 / 정책 / 상담사 응대 / 외부 사유 (4개 중 택 1)
  3. **사실 관계 정리**: 발생 일시 / 관련 카드번호 / 금액 / 회수
  4. **고객 요구사항**: 환불 / 사과 / 정정 / 정책 변경 등 명확히 기록
  5. **이관 / 처리 경로**: 자체 처리 / 부서 이관 / 담당자 / SLA 시일
- **표준 발화 예시**:
  - "고객님 말씀 정리하면, MM월 DD일 OO 카드 결제 OOO원 부분에 대한 OO 사유로 OO 요청 주신 것 맞으실까요?"
  - "이 건은 VOC 카테고리상 'OO/OO' 로 분류되고, 처리 부서는 OO팀입니다. 통상 영업일 N일 이내 결과 안내드립니다."
- **복창 의무**: 분류 라벨과 처리 경로는 고객에게 반드시 복창
- **부분 점수 트리거**:
  - 5요소 중 1개 누락: 10점
  - 5요소 중 2개 누락 또는 카테고리 오분류: 5점
  - 3개 이상 누락 또는 카테고리 오분류 미정정: 0점

## VOC 오분류 즉시 정정 표준

민원 카테고리를 잘못 분류한 경우 통화 종료 전 정정 시 부분 점수 (5점) 인정.

- **정정 트리거**:
  - 본인 인지 (예: 고객 진술이 추가되어 카테고리 재분류 필요)
  - 고객 의문 제기 ("그게 그 분류인가요?")
- **정정 표준 발화**:
  - "방금 안내드린 분류는 OO 였는데 추가 말씀 주신 부분 반영하면 정확히는 OO 로 분류되어야 합니다."
  - "정정 분류 기준 처리 경로는 OO팀이며 SLA 영업일 N일입니다. 다시 안내드려도 될까요?"
- **정정 미시도 시**:
  - VOC 시스템에 잘못된 카테고리 등재 → 부서 이관 잘못됨 → 처리 지연 / 재민원 발생
  - QA Layer 3 Override 입력에 `has_misclassification=True, correction_attempted=False` 마킹
- **정정 후 처리**:
  - VOC 시스템 재등재 (오분류 사실 기록)
  - 사후 안내 SMS 발송 (정확한 분류 / 처리 경로 / SLA 명시)
