## 카드 상품 약관 주요 항목

상담사가 카드 발급 시 반드시 사전에 고지해야 할 약관 핵심 사항이다.

- **연회비**:
  - 국내전용: 카드별 5,000~30,000원 (Deep Dream 카드 기준 15,000원)
  - 해외겸용: 국내전용 + 5,000~10,000원 가산 (Deep Dream 기준 20,000원)
  - 면제 조건: 카드별 상이. 일반적으로 전년 결제 실적 N백만원 이상 시 면제 (상품별 약관 확인 필수)
  - 청구 시점: 발급일 기준 12개월 후 자동 청구

- **이자율**:
  - **일시불 결제**: 무이자
  - **할부 (2개월 이상)**: 연 5.9% ~ 17.9% (개월 수 / 신용평가 차등)
  - **카드론 / 현금서비스**: 연 4.9% ~ 19.9% (한도 내 사용 시점 신용평가 기준)
  - **연체 이자**: 약정 이자율 + 가산 (최대 연 20% 법정 상한)

- **포인트 / 혜택**:
  - 포인트 적립률 / 사용처 / 소멸 기한 (통상 5년) 설명 필수
  - 카드별 핵심 혜택 (캐시백 / 할인 / 마일리지) 1~2개만 강조 — 모든 혜택 나열 금지

## 가족카드 발급 시 배우자/가족 동의 절차

여신금융 감독규정 제15조에 따른 의무 절차이다. 위반 시 약관규제법 / 감독규정 동시 위반.

- **발급 가능 가족 범위**:
  - 배우자 / 직계존속(부모) / 직계비속(자녀, 만 19세 이상) / 형제자매
  - 만 19세 미만 자녀: 학생증 / 보호자 동의 + 별도 약관

- **본인 명시적 동의 절차**:
  1. 가족(배우자 등) 본인 직접 통화 또는 비대면 본인인증 (모바일 인증 / 영상 통화)
  2. 가족카드 약관 / 한도 공유 / 책임 범위 / 분실 시 의무 직접 안내 후 동의 확인
  3. 주카드 회원의 대리 동의로 처리 **금지** (감독규정 위반)

- **고지 4요소**:
  - 한도: 주카드 한도 내 공유 (별도 한도 부여 불가)
  - 책임: 가족카드 사용 금액은 주카드 회원에게 청구
  - 분실 신고: 가족 본인이 즉시 1544-7000 신고 의무
  - 해지: 주카드 회원 또는 가족 본인 모두 해지 가능

## 신용정보 조회 동의 (CB / NICE) 안내 의무

카드 발급 심사 시 KCB 또는 NICE 신용평가사에 신용정보 조회를 위한 본인 명시적 동의가 필수.

- **동의 받아야 할 항목**:
  - 신용평가사 명시: KCB (코리아크레딧뷰로) / NICE 평가정보 (둘 중 하나 또는 둘 다)
  - 조회 목적: 카드 발급 심사 / 한도 산정 / 부정 사용 모니터링
  - 조회 정보: 신용평점 / 연체 이력 / 다른 금융사 대출 잔액 / 카드 발급 이력

- **개인정보 처리 동의 (선택/필수 분리)**:
  - **필수**: 카드 발급 / 결제 / 부가서비스 운영 (거부 시 발급 불가)
  - **선택**: 마케팅 활용 (이메일 / SMS / 푸시 알림) — 비동의 가능
  - 선택 동의를 사실상 강제 처리하면 약관규제법 위반 (\"일단 동의로 진행\" 등)

- **약관 송부 의무**:
  - 약관 전문은 카드 발급 결정 후 SMS 링크 / 이메일 / 카드 동봉 책자로 송부
  - 가입 후 7일 이내 청약철회 가능 안내 필수

## 배송 시스템 (영업일 / 배송지 변경 / 본인 수령)

- **배송 절차**:
  - 발급 결정 → 카드 인쇄 (영업일 1일) → 등기우편 출고 (영업일 1~2일) → 배송 (영업일 1~2일)
  - 총 소요: 영업일 기준 3~5일 (최장 7일)
  - 배송 업체: 우체국 등기 (KT&S 우편물류) — 신한카드는 직접 배송 운영하지 않음

- **카드수령지**:
  - 신청 시 등록한 주소 (집 / 직장 중 선택)
  - 발급 결정 후 수령지 변경 불가 (반송 후 재발급 — 추가 영업일 3~5일)
  - 안내 시 배송 주소 마스킹 필수 (예: \"마포구 *** 동\")

- **본인 수령 원칙**:
  - 수령자 본인 신분증 확인 후 인계 (가족 수령 불가 — 등기 본인배달)
  - 부재 시: 우체국 보관 (3일) / 재배달 신청 / 직접 수령 가능

- **분실 / 미배송 처리**:
  - 출고 후 영업일 5일 경과 미수령: 우체국 추적 → 재발급 절차
  - 재발급 시 추가 비용 무료 (배송 누락 책임 시), 본인 분실 시 5,000원 수수료

## 부가서비스 자동가입 정책 및 안내 의무

- **자동가입 정책**:
  - **신한카드 표준 카드 발급 시 자동가입되는 부가서비스 없음** (감독규정 강제)
  - 보험 / 멤버십 / 컨시어지 등은 **별도 동의 후 가입** 필수
  - 가입 시 월 회비 / 환급 조건 / 해지 절차 사전 안내 필수

- **흔히 오해되는 사항**:
  - \"카드와 같이 자동으로 가입된다\" 안내는 약관규제법 위반 (예시: synthetic_rev_023 케이스)
  - 부가서비스 별도 약관 / 청약철회 권리 (7일) 별도 안내

- **해지 절차**:
  - 1544-7000 또는 신한플레이 앱 → 부가서비스 해지
  - 위약금 없음 (단, 일부 보험 상품은 약정 기간 위약금 가능 — 가입 시 안내)

## 발급·배송 안내 5요소 표준 스크립트 (#905 직결)

상담사는 발급·배송 안내 시 다음 5요소를 **누락 없이 + 숫자/주소 정확히** 안내해야 한다. QA #905 정확한 안내 ★ 만점 기준.

- **5요소 체크리스트**:
  1. **수령지**: "OO시 OO구 ***" — 카드 마스터 등록 주소와 일치 (마스킹 적용)
  2. **배송 예정일**: "MM월 DD일 (요일) 도착 예정" — 영업일 기준 (주말/공휴일 제외)
  3. **연회비**: "연회비 OOO원 (1년차 면제 / 매년 청구일 D-7 사전 통지)"
  4. **이자율**: "할부 수수료 N.N%~M.M% (구간별 차등)" / "현금서비스 N.N%"
  5. **발급조건**: "연소득 OOO원 / 신용평점 OOO점 이상 / 발급 영업일 D~D+3"
- **표준 발화 예시**:
  - "고객님 신청하신 OO카드는 등록 주소 OO시 OO구 *** 으로 MM월 DD일 도착 예정입니다."
  - "연회비는 OOO원이고 발급 후 첫 1년은 면제, 다음 해부터 청구일 7일 전 사전 안내드립니다."
  - "할부 수수료는 12개월 기준 약 N.N% 수준, 정확한 이율은 신청서·이용약관에 안내되어 있습니다."
- **복창 의무**: 고객이 수령지·연회비·이자율·도착일 중 하나를 다시 물어보면 그대로 복창
- **부분 점수 트리거**:
  - 5요소 중 1개 누락: 10점
  - 5요소 중 2개 누락 또는 1개 오안내: 5점
  - 3개 이상 누락 또는 오안내 미정정: 0점

## 발급·배송 오안내 즉시 정정 표준

상담 중 수령지/도착일/연회비/이자율 등을 잘못 안내한 경우 즉시 정정 시 부분 점수 (5점) 인정.

- **정정 트리거**: 본인 인지 또는 고객 의문 제기 ("어 그래요?", "확실해요?")
- **정정 표준 발화**:
  - "죄송합니다, 방금 안내 다시 드리겠습니다."
  - "정확히는 연회비 OOO원입니다. 혼동 드려 죄송합니다."
  - 정정 후 5요소 전체 재복창 권장
- **정정 미시도 시**:
  - 잘못된 수령지로 출고 → 반송 + 재발급 → 영업일 추가 5~10일 손실
  - QA Layer 3 Override 입력에 `has_incorrect_guidance=True, correction_attempted=False` 마킹
- **정정 후 처리**:
  - 카드 마스터 / 배송 시스템 재조회 (오안내 사실 기록)
  - SMS / 앱 알림으로 정확 정보 재전송 (이중 안전망)

**source_ref**: shinhan-issuance-correction v1.0
