# 개인정보보호 상담 지침

## 1. 개인정보 수집/이용 동의 절차

### 상담 시 수집하는 개인정보
- 성명, 생년월일, 연락처
- 계약번호, 보험 상품 정보
- 상담 내용 (녹취 포함)

### 동의 절차
1. 상담 시작 시 녹취 안내: "이 통화는 상담 품질 향상을 위해 녹음됩니다"
2. 개인정보 수집 목적 안내: "상담 처리를 위해 개인정보를 확인하겠습니다"
3. 고객 동의 확인

## 2. 제3자 상담 시 정보 제공 기준

### 절대 제공 불가 정보 (제3자에게)
- 계약 세부 내용 (보장 내용, 보험료, 환급금)
- 개인 연락처, 주소
- 주민등록번호, 계좌번호
- 보험금 청구/지급 내역

### 제공 가능 정보 (제3자에게)
- 일반적인 상품 안내 (공개된 정보)
- 일반적인 절차 안내
- 고객센터 연락처, 운영시간

### 가족/법정대리인 상담
- 관계 확인 필수 (배우자, 부모, 자녀)
- 위임장 또는 동의서 확인
- 제공 범위: 일반 계약 정보까지 (세부 금융 정보 불가)

## 3. 개인정보 유출 방지 수칙

### 화면 보안
- 고객 정보 화면 이석 시 화면 잠금
- 타 상담사에게 고객 정보 화면 노출 금지
- 고객 정보 메모지 파기 의무

### 통화 중 보안
- 고객 개인정보 큰 소리로 읽지 않기
- 주민번호 전체를 말하지 않기 ("앞 6자리만 확인하겠습니다")
- 고객 옆에 제3자 존재 여부 확인

### 사고 발생 시
1. 즉시 팀장에게 보고
2. 유출 범위 파악
3. 고객 통지 (24시간 이내)
4. 개인정보보호 담당부서 이관
5. 사고 보고서 작성

## 4. 녹취 관련 규정

### 녹취 의무 대상
- 보험 계약 체결 관련 상담 (전화판매)
- 보험금 청구 접수
- 계약 변경/해약 요청
- 민원 접수

### 녹취 안내 시점
- 상담 시작 시 (첫인사 직후)
- 자동 안내 멘트가 없는 경우 상담사 직접 안내

### 녹취 보관
- 보관 기간: 계약 종료 후 5년
- 고객 요청 시 녹취 내용 열람 가능
- 녹취 파일 외부 반출 금지
