# 전화 응대 기본 매뉴얼

## 1. 전화 응대 3대 원칙

### 신속성
- 전화벨 3회 이내 수신
- 3회 초과 시 "기다려 주셔서 감사합니다" 사과 멘트 선행
- 보류(Hold) 시간 30초 이내 유지, 초과 시 중간 안내

### 정확성
- 고객 문의에 대한 정확한 정보 제공
- 불확실한 정보는 "확인 후 안내드리겠습니다"로 응대
- 추측성 답변 절대 금지

### 친절성
- 밝고 활기찬 목소리 톤 유지
- 존칭 일관 사용 (~님, ~시죠, ~겠습니다)
- 경어체 사용 의무화

## 2. 전화 응대 표준 프로세스

### 2.1 수신 단계
```
[첫인사] "감사합니다. OO보험 고객센터 OOO입니다. 무엇을 도와드릴까요?"
```
- 소속 + 이름 필수 안내
- 밝고 또렷한 발음
- 속도: 분당 250~280음절 (보통 말하기 속도)

### 2.2 본인확인 단계
```
[본인확인] "본인 확인을 위해 성함과 생년월일을 말씀해 주시겠어요?"
```
- 필수 확인 항목 (2가지 이상):
  1. 성명 (필수)
  2. 생년월일 또는 주민등록번호 앞자리 (필수)
  3. 연락처 (선택)
  4. 계약번호 (선택)
- 제3자 상담 시 관계 확인 및 위임 여부 확인

### 2.3 용건 파악 단계
```
[용건확인] "네, OO 관련 문의이시군요. 자세히 안내해 드리겠습니다."
```
- 고객 발언을 끊지 않고 끝까지 경청
- 핵심 용건 되묻기/확인
- 상담 유형 분류 (청구, 해약, 보장문의, 변경, 일반문의)

### 2.4 처리 단계
- 문의 유형에 맞는 처리 절차 수행
- 필수 안내사항 빠짐없이 전달
- 법적 고지사항 수행 (해당 시)
- 처리 결과 안내

### 2.5 마무리 단계
```
[내용정리] "오늘 안내해 드린 내용 정리해 드리겠습니다. ..."
[추가문의] "혹시 더 궁금하신 사항 있으시나요?"
[끝인사] "감사합니다. OO보험 OOO이었습니다. 좋은 하루 되세요."
```

## 3. 금지 표현 및 대체 표현

| 금지 표현 | 대체 표현 |
|----------|----------|
| 모릅니다 | 확인해 보겠습니다 |
| 안 됩니다 | ~하시면 가능합니다 |
| 그건 저희 쪽 일이 아닙니다 | 담당 부서로 연결해 드리겠습니다 |
| 아까 말씀드렸잖아요 | 다시 한 번 안내해 드리겠습니다 |
| 잠시만요 (이유 없이) | 확인을 위해 잠시만 기다려 주시겠어요? |
| 네? 뭐라고요? | 죄송합니다, 다시 한 번 말씀해 주시겠어요? |

## 4. 보류(Hold) 응대 기준

- 보류 전: "확인을 위해 잠시만 기다려 주시겠습니까? 약 OO초 정도 소요됩니다."
- 보류 중 (30초 초과 시): "조금만 더 기다려 주시겠습니까? 확인 중입니다."
- 보류 해제: "기다려 주셔서 감사합니다."
- 보류 최대 시간: 2분 초과 시 콜백 안내

## 5. 이관(전환) 기준

- 이관 사유 고객에게 설명
- 이관 대상 부서/담당자 안내
- 이관 시 이전 상담 내용 인수인계
- "담당 부서로 연결해 드리겠습니다. 지금까지의 상담 내용은 함께 전달됩니다."
