# 불만 고객 응대 가이드라인

## 1. 불만 고객 유형별 응대 전략

### Type A: 감정형 (화가 난 고객)
- **특징**: 큰 소리, 반복적 불만 표출, 감정적 언어 사용
- **응대 전략**:
  1. 절대 고객 말 끊지 않기 — 감정이 가라앉을 때까지 경청
  2. 공감 표현: "불편을 드려 정말 죄송합니다"
  3. 감정 인정: "화가 나실 만합니다"
  4. 해결 의지 표현: "제가 바로 확인해서 도와드리겠습니다"
  5. 구체적 해결 방안 제시

### Type B: 논리형 (따지는 고객)
- **특징**: 근거 요구, 규정 인용, 비교 분석
- **응대 전략**:
  1. 정확한 근거 제시 (약관 조항, 규정 번호)
  2. 논리적 순서로 설명
  3. 문서 근거 제공 제안 ("해당 약관 내용 서면으로 보내드릴까요?")
  4. 인정할 부분은 인정 ("말씀하신 부분이 맞습니다")

### Type C: 보상형 (보상을 원하는 고객)
- **특징**: 금전적 보상 요구, 상위자 연결 요구
- **응대 전략**:
  1. 고객의 요구 정확히 파악
  2. 가능한 보상 범위 안내
  3. 불가능한 요구 시 대안 제시
  4. 에스컬레이션 기준에 따라 상위자 연결

## 2. 에스컬레이션 3단계 절차

### 1단계: 상담사 자체 해결
- 사과 + 원인 설명 + 해결 방안 제시
- 상담사 권한 내 즉시 처리 가능한 사안

### 2단계: 팀장/수퍼바이저 연결
- 조건: 고객 요청 시 또는 상담사 권한 초과 시
- 절차:
  1. 고객에게 연결 안내: "담당 팀장님께 연결해 드리겠습니다"
  2. 이전 상담 내용 인수인계 (메모 전달)
  3. 고객 대기 시간 안내
  4. 팀장 부재 시 콜백 일정 안내

### 3단계: 민원 접수 및 전담부서 이관
- 조건: 반복 불만, 법적 이슈, 2단계 미해결
- 절차:
  1. 민원 접수 번호 발행 및 안내
  2. 처리 예상 기간 안내 (통상 7~14영업일)
  3. 진행 상황 안내 방법 선택 (전화/SMS/이메일)
  4. 금융감독원 민원 신청 방법 안내 (의무)

## 3. 절대 금지 행동

1. 고객 말 끊기 (감정 고조 시 특히 주의)
2. 고객 탓하기 ("고객님이 잘못 이해하신 겁니다")
3. 회피 ("저는 담당이 아닙니다")
4. 무관심 ("원래 그렇습니다")
5. 역감정 표출 (목소리 높이기, 한숨)
6. 개인적 의견 표출 ("저도 그게 이상하다고 생각합니다")

## 4. 불만 응대 핵심 화법

### 공감 3단계 화법
1. **인정**: "그런 상황이셨군요"
2. **공감**: "불편하셨겠습니다 / 걱정되셨겠습니다"
3. **의지**: "제가 확실히 확인해서 안내드리겠습니다"

### 쿠션 화법 (부정적 답변 시)
- "말씀하신 부분은 충분히 이해합니다. 다만, ..."
- "고객님의 마음은 이해하지만, 현재 규정상 ..."
- "아쉽게도 해당 부분은 어려우나, 대신 ..."
