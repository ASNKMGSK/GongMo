# 보험 상담 필수 프로세스 가이드

## 1. 보험금 청구 상담 프로세스

### 필수 확인 사항
1. 청구인 본인확인 (성명 + 생년월일 + 계약번호)
2. 청구 사유 확인 (질병/상해/사망/만기 등)
3. 사고 발생일 확인
4. 현재 치료 상태 확인

### 필수 안내 사항
1. **필요 서류 목록**
   - 보험금 청구서 (자필 서명)
   - 진단서 또는 소견서 (질병/상해)
   - 진료비 영수증 및 세부내역서
   - 사고증명서 (교통사고 시)
   - 사망진단서 (사망보험금)
   - 수익자 신분증 사본

2. **청구 방법 안내**
   - 온라인: 홈페이지 또는 모바일 앱
   - 오프라인: 우편 또는 지점 방문
   - 팩스: 02-XXXX-XXXX

3. **처리 기간**
   - 접수 후 3영업일 이내 심사
   - 보험금 지급: 심사 완료 후 3영업일 이내
   - 추가 서류 요청 시 별도 안내

4. **지급 계좌 확인**
   - 계약자/수익자 본인 명의 계좌만 가능
   - 타인 명의 계좌 지급 불가

### 주의사항
- 보험금 부지급 사유 설명 의무 (면책, 고지의무 위반 등)
- 이의 제기 절차 안내 (분쟁조정위원회, 금융감독원)
- 시효 안내: 보험금 청구 시효 3년

## 2. 보험 해약 상담 프로세스

### 해약 전 필수 고지 사항 (법적 의무)
1. **해약 환급금 안내** — 현재 기준 정확한 금액
2. **불이익 사항 고지**
   - 보장 즉시 소멸
   - 재가입 시 보험료 인상 (연령 증가, 건강 상태 변화)
   - 면책기간 재적용
   - 납입 원금 대비 환급금 손실 가능성
3. **대안 제시**
   - 감액 완납: 보험료 납입 중단, 보장 축소 유지
   - 납입 유예: 일정 기간 보험료 납입 면제
   - 보험료 자동대출: 해약환급금 범위 내 자동 대출
   - 보장 축소: 특약 해지로 보험료 절감

### 해약 처리 절차
1. 해약 의사 재확인 ("해약 시 보장이 소멸되는데 진행하시겠습니까?")
2. 본인확인 (강화: 성명 + 생년월일 + 계약번호 + 연락처)
3. 해약 환급금 최종 확인 및 안내
4. 불이익 사항 재고지
5. 환급금 입금 계좌 확인
6. 해약 처리 실행
7. 처리 완료 안내 (입금 예정일, 철회 가능 기간)

### 해약 철회
- 해약 신청 후 30일 이내 철회 가능 (전화 또는 서면)
- 철회 시 기존 계약 조건 동일하게 복원

## 3. 보장 내용 문의 프로세스

### 필수 확인
1. 계약 기본 정보 (상품명, 계약일, 만기일)
2. 주요 보장 항목 및 보장 한도
3. 특약 내용 및 보장 범위

### 필수 안내
1. 면책 사항 (보장 제외 항목)
2. 감액 기간 (가입 후 1~2년)
3. 갱신 조건 (갱신형의 경우)
4. 보험료 변동 가능성

### 주의사항
- "약관에 의거하여" 정확한 정보만 안내
- 불확실한 보장 여부는 심사 후 안내
- 과장/축소 안내 금지

## 4. 계약 변경 프로세스

### 변경 가능 항목
- 수익자 변경
- 연락처/주소 변경
- 보험료 납입 방법 변경
- 특약 추가/해지

### 변경 불가 항목
- 계약자 변경 (승계 절차 별도)
- 보험 기간 변경
- 기본 보장 내용 변경

### 변경 처리 기준
- 단순 변경 (연락처, 주소): 즉시 처리
- 보장 변경 (특약 추가/해지): 심사 후 처리 (1~3영업일)
- 수익자 변경: 서류 접수 후 처리
