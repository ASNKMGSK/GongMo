# QA 모니터링 평가표 작성 가이드

## 1. 평가표 구성 원칙

### 평가 영역 (5대 영역)
1. **오프닝** (Opening) — 첫인사, 본인확인, 용건 파악
2. **상담 스킬** (Consulting Skill) — 경청, 공감, 설명력
3. **업무 처리** (Task Processing) — 정확성, 프로세스 준수, 필수안내
4. **고객 관계** (Customer Relations) — 존칭, 공감 표현, 불만 대응
5. **클로징** (Closing) — 내용 정리, 추가 문의, 끝인사

### 배점 설계 원칙
- 총점: 100점 만점 (또는 업체별 자체 기준)
- 필수 항목: 미수행 시 0점 처리 (인사, 본인확인, 법적 고지)
- 가감점 항목: 기본 점수에서 ± 조정
- 치명적 오류: 관련 영역 전체 0점 (개인정보 유출, 허위 안내 등)

## 2. 영역별 세부 평가 항목

### 2.1 오프닝 (20점)

| 항목 | 배점 | 평가 기준 |
|------|------|----------|
| 첫인사 | 5점 | 규정 멘트 사용 (소속+이름), 밝은 톤 |
| 본인확인 | 10점 | 2개 이상 확인항목 수행, 절차 정확성 |
| 용건 파악 | 5점 | 고객 문의 내용 정확 확인, 상담 유형 분류 |

### 2.2 상담 스킬 (25점)

| 항목 | 배점 | 평가 기준 |
|------|------|----------|
| 적극적 경청 | 8점 | 고객 말 끊지 않음, 호응 표현, 메모 |
| 설명력 | 10점 | 전문용어 쉬운 설명, 논리적 순서, 핵심 강조 |
| 상황 파악 | 7점 | 고객 니즈 정확 파악, 맥락 이해, 적절한 질문 |

### 2.3 업무 처리 (25점)

| 항목 | 배점 | 평가 기준 |
|------|------|----------|
| 정보 정확성 | 10점 | 상품/서비스 정보 정확, 가격/조건 오류 없음 |
| 프로세스 준수 | 8점 | 필수 절차 수행, 스크립트 준수 |
| 필수 안내 | 7점 | 법적 고지, 약관 안내, 동의 절차 |

### 2.4 고객 관계 (20점)

| 항목 | 배점 | 평가 기준 |
|------|------|----------|
| 존칭/매너 | 5점 | 존칭 일관 사용, 공손한 어미 |
| 공감 표현 | 8점 | 감정 인정, 상황 공감, 진정성 |
| 불만 대응 | 7점 | 사과 표현, 해결 의지, 에스컬레이션 |

### 2.5 클로징 (10점)

| 항목 | 배점 | 평가 기준 |
|------|------|----------|
| 내용 정리 | 3점 | 처리 결과 요약, 후속 절차 안내 |
| 추가 문의 | 3점 | "더 궁금하신 사항" 확인 |
| 끝인사 | 4점 | 규정 멘트 사용 (소속+이름), 감사 인사 |

## 3. 치명적 오류 (Fatal Error) 항목

아래 항목 발생 시 해당 영역 **즉시 0점 처리**:

1. **개인정보 유출** — 제3자에게 고객 개인정보 안내
2. **허위/거짓 안내** — 사실과 다른 정보 의도적 제공
3. **고객 비하/모욕** — 반말, 비하 발언, 고압적 태도
4. **업무 거부** — 정당한 사유 없이 처리 거부
5. **본인확인 완전 미수행** — 본인확인 절차 전체 생략

## 4. 평가 등급 체계

| 등급 | 점수 범위 | 의미 |
|------|----------|------|
| S (Excellent) | 95~100 | 모범 상담, 코칭 사례 활용 |
| A (Good) | 85~94 | 우수 상담 |
| B (Average) | 75~84 | 보통 수준, 개선점 있음 |
| C (Below) | 65~74 | 미흡, 집중 코칭 필요 |
| D (Poor) | 50~64 | 불량, 재교육 필요 |
| F (Fail) | 0~49 | 심각한 문제, 즉시 조치 |

## 5. 모니터링 방법

### 5.1 실시간 모니터링 (Side-by-Side)
- QA 담당자가 상담사 옆에서 직접 청취
- 즉각적 피드백 가능
- 월 1회 이상 권장

### 5.2 녹취 모니터링 (Recording Review)
- 녹취된 상담 통화를 사후 청취/평가
- 가장 보편적인 방법
- 월 5~10콜 이상 샘플링 권장

### 5.3 미스터리 콜 (Mystery Call)
- 가상 고객으로 전화하여 평가
- 실제 응대 수준 측정
- 분기 1회 권장
